Produktbildspel

Kommersiellt plugin till e-handelsplattformen Tictail som gör det möjligt att visa utvalda produkter i ett bildspel. Pluginet byggdes i PHP / Symfony och Javascript med integration mot Tictails API.

Jag använde från början jQuery UI:s funktion för drag&drop för att kunna byta ordning i bildspelet, men den visade sig vara undermålig så jag byggde en egen i stället.

Länk: https://tictail.com/apps/featured-products
