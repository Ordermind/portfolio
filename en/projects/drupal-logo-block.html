Logo Block

A logo is present on almost every website, but in Drupal the common way that the logo is displayed is relatively static, with few possibilities for customizations. Usually it is rendered by the theme in a pre-defined region, links to the front page and uses the full image without any support for automatic image editing in the form of image styles.

Logo Block solves these problems through a block that you can place freely on the page, with a configuration form where you can choose whether the logo should link to a page (and if so which page), and also select an image style to make sure that the design looks good even if the logo is replaced in the future.

Link: https://www.drupal.org/project/logo_block
