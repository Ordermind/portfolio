Product slideshow

Commercial plugin for the e-commerce platform Tictail that makes it possible to display selected products in a slideshow. The plugin was built in PHP / Symfony and Javascript and integrates with Tictail’s own API.

In the beginning I used jQuery UI:s function for drag&drop to enable users to switch image order in the slideshow, but it turned out to be of poor quality so I built my own instead.

Link: https://tictail.com/apps/featured-products
