Wärnelius Buss

Wärnelius Buss has a system for administering leisure trips that is built in .NET. They needed a website that was integrated to that system so that they could display and book the trips on the website but keep all of the administration inside the .NET system. I built the backend and frontend in PHP / Drupal 7 with a REST API that connected to the .NET system in order to periodically update the trips on the website.
